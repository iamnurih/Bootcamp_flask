# 1일차 [실습 1]: Flask 기본 코드 구조

* 윈도우
  ```
  # 가상환경 만들기
  python -m venv venv

  # 가상환경 실행하기
  venv\Scripts\activate.bat

  # flask 모듈 설치하기
  pip install flask
  ```

* 리눅스 / 맥
  ```
  # 가상환경 만들기
  python3 -m venv venv

  # 가상환경 실행하기
  source venv/bin/activate

  # flask 모듈 설치하기
  pip install flask
  ```
